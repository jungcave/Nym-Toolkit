//
// PROPERTIES

Axis bend {n} [v] - Setup:
- target object and add geometry
- [v] create an empty and move it to object's pivot
- [v] parent empty to object
- [v] add simple deform [bend] modifier
- [v] set empty as origin and choose bend axis
- rotate empty on chosen axis 90deg
- set chosen axis and bend angle in modifier settings

Radial array {n} [v] - Setup:
- target object and move its pivot to future rotation origin
- [v] apply object scale
- [v] create an empty and move it to object's pivot
- [v] parent empty to object
- [v] add to object array modifier with [_] radial offset and [v] object offset
- [v] set empty as object offset
- rotate empty on chosen axis until it fits
- increase array count in modifier settings

Overlay plane on rounded area {TexTools}:
- target object
- in edit mode separate object's rounded area and unwrap it
- in uv editor [textools] rectify unwrapped area and create uv mesh for it
- in object mode put overlay plane on top of the created uv mesh
- parent uv mesh to plane
- add surface deform modifier to the plane and set target to the uv mesh, bind
- in data attrs change uv mesh shape key value from 1 to 0
- hide uv mesh
